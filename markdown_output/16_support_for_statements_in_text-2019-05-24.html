timestamps <- timestamps_raw %>%

ts <- timestamps %>% 
  mutate(duration = end - start)

read_counts_for_merge <- read_counts %>%
  select(sample_id, parent_id, measurement, read_count) %>%
  spread(measurement, read_count)

ts_anno <- left_join(ts, read_counts_for_merge) %>% 
  group_by(pipeline) %>%
  mutate(minimum_pipeline_time = median(as.numeric(duration)[grepl("_est1M", sample_id)]),
         adjusted_duration = as.numeric(duration) - minimum_pipeline_time,
         min_per_M_total_reads = adjusted_duration / (total_sequences / 1e6 - 1),
         min_per_M_MEND_reads = adjusted_duration / (MEND / 1e6 - 1))

  summarize(median_min_per_M_total_reads = median(min_per_M_total_reads),
            median_min_per_M_MEND_reads = median(min_per_M_MEND_reads),

dynamically generated result

Based on the processing times of 95 subsamples, the expression quantification pipeline takes 28.4 minutes for every RNA-Seq sample plus 6.7 minutes per million total reads present in the data.

The MEND qc quantification pipeline takes 1.6 minutes for every RNA-Seq sample plus 2.1 minutes per million total reads present in the data.

For a sample with 70 M total reads as recommended above, the expression pipeline would be predicted to take to take 497 minutes (8.3 hours). To add MEND qc would increase the duration by 149 minutes (2.5 hours).

We give this number in total reads because the number of MEND reads cant be known until the pipeline is run, and is therefore not available for estimating how long the pipeline will take.
